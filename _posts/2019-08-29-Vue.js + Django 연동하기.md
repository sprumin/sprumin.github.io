# Vue.js + Django

[주의! 부정확한 내용이 있을 수 있습니다. 댓글 또는 이메일로 피드백 해주시면 감사드리겠습니다.]

이번 포스팅에서는 Vue.js 와 Django 를 설치하고 서로 연동하여 Django 서버에서 Vue.js 메인 페이지를 띄워볼것이다.

우선 Django 를 먼저 설치하고 Django 프로젝트 내에 Frontend 라는 이름으로 vue-cli 를 설치할 예정이다.

<br/>

### Django 설치

python 3.7.3, Windows 7 x64 기준으로 작성되었습니다.

- `pip install virtualenv` : 파이썬 가상환경 설치
- `virtualenv venv` : venv 라는 이름의 가상환경 생성
- `venv\scripts\activate` : 가상환경 접속
  - 리눅스 환경에선 source bin/activate...?였나

가상환경을 세팅해주는 이유는 기존 로컬에서 개발 시 사용되던 모듈들과 이 프로젝트에서 필요한 모듈들의 버전이 다를 수 있고, 배포 시 불필요한 모듈들이 requirements.txt 에 포함되지않게 하기위해서라고 생각한다. 한마디로 프로젝트 관리가 편하다고 말하면 될듯하다.

- `pip install django` : Django 설치
- `django-admin startproject RuminScheduler` : RuminScheduler 라는 Django 프로젝트 생성
- `cd RuminScheduler` : Django 프로젝트 폴더로 이동
- `python manage.py makemigrations` : database migration 파일 생성
- ` python manage.py migrate` : database migration
- `python manage.py runserver` : Django 서버 실행

### Vue.js 설치

Django 서버와 Vue.js 서버를 동시에 실행해놓아야 하기때문에 Vue.js 를 설치하고 실행할 Command 창을 하나 더 열어주고 위 Django 프로젝트 폴더에서 가상환경 접속까지 진행해준다.

Vue.js 는 npm 명령어를 통해 설치해야하는데 npm 명령어를 사용하기 위해 [node.js](https://nodejs.org/ko/download/) 를 설치해주도록 하자. ( 현재 폴더 위치는 Django 프로젝트 폴더 )

- `npm install -g vue-cli` : Vue.js 설치
- `vue init webpack-simple frontend` : frontend 라는 Vue.js 프로젝트(?) 생성
- `cd frontend` : frontend 폴더로 이동
- `npm install` : 실행에 필요한 package 파일 install
- `npm run dev` : Vue.js 서버 실행

### Vue.js + Django 연동

둘 다 설치가 끝났으니 webpack 을 이용하여 서로 연동시켜주는 작업을 할 것이다.



**[ Django 실행중인 커맨드 창 ]**

Ctrl + C 를 입력하여 서버를 종료한 후 `pip install django-webpack-loader` 명령어를 입력하여 설치한다.

그후 `프로젝트이름/settings.py` 파일을 수정해줄것이다.

- `INSTALLED_APPS` : 리스트안에 'webpack_loader' 를 추가
- `TEMPLATES` : 리스트안에 'DIRS' 에 `os.path.join(BASE_DIR), "templates"` 를 추가
  - templates 을 호출할 위치를 지정해주는 코드이다.

```python
# WEBPACK
WEBPACK_LOADER = {
    'DEFAULT': {
        'CACHE': not DEBUG,
        'BUNDLE_DIR_NAME': '',
        'STATS_FILE': os.path.join(BASE_DIR, 'frontend/webpack-stats.json'),
        'POLL_INTERVAL': 0.1,
        'TIMEOUT': None,
        'IGNORE': ['.+\.hot-update.js', '.+\.map']
    }
}	
```

파일 하단에 추가해준다. 각 값에 대한 상세한 내용은 검색바랍니ㄷ.. 저도 공부중이라..



**[ Vue.js 실행중인 커맨드 창 ]**

Vue.js 에서도 연동을 위해 webpack 관련 모듈들을 설치해준다.

`npm install --save-dev webpack-bundle-tracker write-file-webpack-plugin` 명령어 실행 후 `webpack.config.js`  파일을 아래와 같이 수정해준다.

```javascript
# 상단 모듈선언 부분에 추가
var BundleTracker = require('webpack-bundle-tracker')
var WriteFilePlugin = require('write-file-webpack-plugin')

# module.exports 안에 추가
# 어디 적어야할지 모르겠다면 module: {~~~}, 이 이후에 작성
plugins: [
    new BundleTracker({
        filename: 'webpack-stats.json'
    }),
    new WriteFilePlugin()
]
```

저장한 뒤 `npm run dev` 명령어로 Vue.js 서버를 실행하고 메인 페이지가 뜨는것을 확인한 후

`webpack-stats.json` 파일이 생성되었는지 확인한다.

<br/>

다시 `webpack.config.js` 파일로 이동해서 아래와 같이 수정한다.

```javascript
# 상단 module.export -> output 안에 publicPath 를 수정
publicPath = 'http://localhost:8080/dist/',
```

django 프로젝트 폴더 하위에 templates 라는 폴더를 생성 후 index.html 파일을 하나 만들어 아래 처럼 작성한다.

```html
<!DOCTYPE html>
<html>
   	<head>
        {% load render_bundle from webpack_loader %}
        <meta charset='utf-8'>
        <title>Title</title>
    </head>
    <body>
        <div id="app">
        </div>
        
        {% render_bundle 'main' %}
    </body>
</html>
```

Vue.js 에서 설정한 webpack 파일을 django 템플릿에서 불러오는내용으로 보인다.

그 후 `urls.py` 에서 해당 템플릿을 route 해준다.

```python
# 상단에 추가
from django.views.generic import TemplateView

# urlpatterns 에 추가
path('', TemplateView.as_view(template_name='index.html')),
```

이제 설정이 끝났으니 각 커맨드 창에서 Vue.js 와 Django 서버를 실행시킨 뒤 `127.0.0.1:8000` 또는 `localhost:8000` ( 8000 포트는 django 서버의 default 포트 ) 로 접속했을 때 Vue.js 메인페이지가 나타난다면 성공적으로 연동을 마친것이다.
