# 레드벨벳 슬기 팬 페이지

-  프로젝트도할겸 덕질도할겸 레드벨벳 슬기 사진을 모아 웹서버를 구축해보기로했다.
-  실제 사용중인 소스는 [ Github ](https://gibhub.com/sprumin/LoveSeulgi) 에서 볼 수 있다.
-  크롤러와 같이 개발하여 첫 Django 프로젝트를 진행해본 결과 부족한점이 너무나도 많았고 공부하는 방법과 포스팅을
   하는 이유를 얻지 못하였기에 해당 프로젝트의 포스팅은 여기서 마치도록하고 추후 해당 프로젝트의 버그 수정 도중
   발생한 이슈에 대해서는 기억하기 위해 포스팅 될 수 있다.
