# Vue.js + Django 로 개발하는 일정관리 웹 서비스
[Github Link](https://github.com/sprumin/Vue-Django-Scheduler)

-  웹 백엔드 개발을 쭉 해왔지만 다른 프론트엔드 개발자, 웹 프론트엔드 프레임워크와 같이 개발해본적이 없다.
-  그렇다고 현재 회사에서 프론트엔드 개발자가 있지도 않아 혼자 Vue.js 공부도 할 겸해서 진행하게되었다.
-  이전 프로젝트 포스팅 방식과는 다르게 프로젝트 진행도중 기억해야할 부분이 있으면 숙지 후 바로 포스팅할 예정이다.
